# 🍽️ Site Web Officiel — Restaurant Le Délice

Bienvenue sur le dépôt du projet web pour le restaurant "Le Délice", une enseigne célébrant la richesse, la générosité et la tradition de la gastronomie haïtienne.

## 📌 Présentation du Projet

Ce projet consiste en la création d'un site web vitrine complet, responsive et moderne pour le restaurant "Le Délice". Il permet aux visiteurs de découvrir l'histoire du restaurant, de consulter la carte interactive des plats, d'explorer la galerie visuelle et d'effectuer des réservations ou des demandes de contact en ligne.

### 🌟 Fonctionnalités Principales

- Accueil (`index.html`) : Présentation générale, aperçu de l'histoire, chiffrages clés / statistiques, plats signatures et incitation à la réservation.
- À propos (`A-propos.html`) : Histoire du restaurant (racines familiales à Jacmel), valeurs fondamentales (Authenticité, Hospitalité, Fraîcheur) et présentation de l'équipe.
- Menu (`Menu.html`) : Carte complète filtrable par catégories (Entrées, Plats principaux, Pizza, Accompagnements, Desserts, Boissons) avec prix en HTG.
- Galerie (`Galerie.html`) : Grille visuelle dynamique mettant en valeur les plats emblématiques de la maison (Diri Djondjon, Akra, Bannan Peze, Tassot, etc.).
- Réservation (`Réservation.html`) : Formulaire de réservation complet (nom, email, téléphone, date, heure, nombre de convives, occasion).
- Contact (`Contact.html`) : Formulaire de demande d'informations, événements de groupe, partenariats ou suggestions.
- Interactivité & UX (`js/script.js`) :
  - Menu burger responsive pour la navigation mobile.
  - Défilement fluide (_Smooth Scroll_) vers la section désirée dans la galerie.
  - Filtrage dynamique par onglets sur la page Menu.
  - Animation de la barre de navigation au défilement (Sticky Header).

## 🛠️ Technologies Utilisées

- HTML5 : Balisage sémantique et accessible.
- CSS3 : Styles globaux, typographies, jeux de couleurs (`css/style.css`) et mise en page responsive (`css/responsive.css`).
- JavaScript (ES6) : Manipulation du DOM et gestion des événements interactifs (`js/script.js`).

Installation & Utilisation

1. Cloner le dépôt Git : git clone https://github.com/julotsara25/Le-Delice.git
2. Accéder au dossier : cd Le-Delice
3. Lancer le site :
   Ouvrez simplement le fichier index.html dans un navigateur web ou utilisez l'extension Live Server dans VS Code pour un rendu dynamique local
